# NodeJS-Quickstart
Basic Node.js web application

## Installation

```sh
npm install   # install dependencies
```

Using MongoDB? It should be running at localhost:27017

## Run

```sh
npm start       # starts the server
npm run babel   # compiles client-side JS + watch changes
```
